Installing and using Plex Media Server

1. Begin by visiting https://www.plex.tv/ and Create an Account. An account with Plex is required to access the Plex Media Server software.

2. After creating your account, scroll to the bottom of the Plex.tv website, and click the Plex Media Server link under downloads.

3. Select Windows as your platform, click Download and open the downloaded file.

4. Open the file you downloaded, and click Install.

  5. Approve any Windows warnings message that may be generated. Figure 5. Allow changes.

6. After a successful installation, click the Launch button when prompted.

a. Or Open the application from the Start Menu.

  7. After a few seconds you’ll be prompted to sign in with the credentials you created in Step 1.

  8. After signing in, Plex will open a browser to your own Plex Media Server! The initial screens provide some information about how the server works. You’ll notice the browser is directing you to URL 127.0.0.1, which tells your computer that it is hosting the webpage. Click GOT IT! Figure 9. Click GOT IT! to proceed.

  9. Click the white X to skip any advertisements for Plex Pass. You can sign up for these later.

10. Untick the box marked Allow me to access my media outside my home. You can enable this feature later on when you're prepared to make a configuration change to your home network’s router. Name your Plex Media Server or accept the default name of PLEX and click NEXT.

11. Steps 11a-d will add media from your computer to Plex’s content library. If your Windows Music, Photos, or Videos folder contain content, Plex will normally choose them for you. You can remove them by clicking the red x at the end of the content library’s name. If you don’t want to share any other media click next.

a. If you’d like to specify content not found by Plex, click Add Library. In the first dialog, specify the type of media the folder contains.

  b. Next, specify the folders location then click add. In this example I’ve chosen the default Windows picture folder for my user account.

c. Skip the advanced configuration by clicking Add Library. You can return to those options later. Figure 15. Add the library.

  d. If you have multiple content libraries you’d like to add, repeat these steps in 9a-c until complete, then click Next. Figure 16. Add more content or click NEXT.

12. Plex will prompt you to get applications for your mobile devices. Skip this step by clicking DONE. Figure 17. Click DONE.

13. Then click FINISH SETUP. Figure 18. Finish setup.

14. Plex Media Server is now installed and running. Media libraries that you’ve added and Plex’s own services will be show to the left side of the screen. The libraries can be reordered. You can see the media that I added earlier shown. Figure 19. View content.

15. While Plex Media Server focuses on organizing your own content, Plex offers free movies and shows via their own streaming service. Click Movies & Shows to see some of Plex’s content. The other menus such as Web Shows and News offer similar fare. Figure 20. View Plex's streaming content
